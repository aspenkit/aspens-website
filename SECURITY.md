# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in aspens, please report it responsibly.

**Email:** Open a private security advisory at [github.com/aspenkit/aspens/security/advisories](https://github.com/aspenkit/aspens/security/advisories)

Do not open a public issue for security vulnerabilities.

## Scope

This repository contains the aspens documentation website (static site). The aspens CLI source code is at [github.com/aspenkit/aspens](https://github.com/aspenkit/aspens).
